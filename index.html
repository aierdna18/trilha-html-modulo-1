Informações básicas

HTML(HiperText Markup Language) - Linguagem de marcação de hipertexto, é o bloco de construção mais básico da internet, define assim o significado e a estrutura do conteúdo sa web. "Hipertexto" refere-se aos links que conectam páginas entre si , seja dentro de um único site ou entre sites. Links são um aspecto fundamental da web. Ao carregar um conteúdo da internet e vinculá-lo a páginas criadas por outras pessoas, você se torna um participante ativo na world web wide.

O HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição em um navegador web. a marcação HTML usa "elementos" especiais, que consistem em nome do elemento entre '<' e - '>' ou '/>'.

Nas seções abaixo serão apresentados estes elementos, que também são chamados de tags.

Citação

O elemento HTML 'blockquote' indica que o texto incluído é uma citação estendida. Normalmente, isso é renderizado visualmente por recuo, este sendo possível ser alterado. Uma URL para a fonte da citação pode ser fornecida usando o atributo cite, enquanto uma representação de texto da fonte pode ser fornecida usando o elemento 'cite'. Exemplo:

O ser diferente condena a uma solidão fatal. - Aldous Huxley

ou

O ser diferente condena a uma solidão fatal.

- Aldous Huxley

Links de referências

A tag 'a', também chamado de âncora, com seu atributo 'href ', cria um hiperlink para as paginas da web, arquivos, endereços de email, ou qualquer coisa que uma URL possa endereçar.

Em conjunto com a tag 'target' define-se o contexto de navegação, sendo _self para padrão, _blank para uma nova guia, _parent para o pai atual(caso não haja, comporta-se como _self), e _top para o contexto mais alto que é o ancestral atual(se não houver comporta-se como _self).
